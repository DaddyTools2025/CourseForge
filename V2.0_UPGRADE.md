## ✨ 新功能

### 1️⃣ 程序内配置 API
**告别手动编辑配置文件！**

- ✅ 首次启动自动进入配置界面
- ✅ 图形化填写 API 信息
- ✅ 实时测试 API 连接
- ✅ 随时修改或删除配置

**操作流程：**
1. 双击 `start.bat` (Windows) 或 `./start.sh` (Mac/Linux)
2. 在配置界面填写 API Key 和 Base URL
3. 点击"测试连接"验证是否正常
4. 点击"保存配置"完成设置
5. 自动进入主界面开始使用

### 2️⃣ 一键启动脚本
**自动处理所有依赖，真正的零配置！**

**新启动脚本功能：**
- ✅ 自动检查 Python 版本
- ✅ 自动创建虚拟环境
- ✅ 自动安装依赖包（使用国内镜像源加速）
- ✅ 自动启动应用
- ✅ 彩色输出，进度清晰

**使用方法：**

**Windows:**
```
双击 start.bat
```

**Mac/Linux:**
```bash
./start.sh
```

就这么简单！脚本会自动完成所有环境配置。

### 3️⃣ 专业系统 Prompt 注入
**内置银行课程设计师专业模板！**

**默认系统 Prompt 特性：**
- 🎓 15年国有银行课程设计经验
- 📚 精通 ADDIE 教学模型和 Bloom 认知分类法
- 🏷️ 严格执行白标输出（自动去除机构名称）
- 🎨 自适应受众级别的语气和深度

**系统 Prompt 内容：**
```
# Role
你是一位拥有15年经验的国有银行资深课程设计师，精通ADDIE教学模型和Bloom认知分类法。

# Constraints (严格执行)
1. **White Label Only**: 输出的所有内容必须是通用的、中立的。
   严禁出现任何"XX银行"、"XX大学"、"XX机构"等名称。
   若必须提及，请用"我行"或"本企业"代替。
2. **Format**: 输出必须结构清晰，Markdown 格式优先。
3. **Tone**: 
   - 对初级员工：鼓励、亲切、简单直接
   - 对中级员工：专业、深度、引导思考

# Task Workflow
1. **Analyze**: 读取课件内容
2. **Adapt**: 根据受众级别调整深度和语气
3. **Generate**: 生成各类教学材料
```

**自定义选项：**
- 💡 可在配置界面勾选"使用自定义系统 Prompt"
- 📝 自由编辑，定制专属教学风格
- 🔄 随时切换默认模板或自定义模板

## 🔄 从 v1.0 升级

### 迁移配置

如果你之前使用的是 v1.0，可以将 `.streamlit/secrets.toml` 中的配置手动填入 v2.0 的配置界面。

**v1.0 配置文件格式：**
```toml
[ai_config]
api_key = "sk-xxx"
base_url = "https://xxx/v1"
model_name = "gemini-2.5-pro"
```

**v2.0 迁移步骤：**
1. 启动 v2.0
2. 在配置界面填入相同的值
3. 保存配置
4. 删除旧的 `.streamlit` 目录（可选）

## 💡 使用技巧

### 技巧 1: 快速测试 API
在配置界面，先点击"测试连接"而不是直接保存，可以避免填写错误的配置。

### 技巧 2: 自定义系统 Prompt
根据你的教学风格调整系统 Prompt：
- 针对技术培训：强调"实操"、"案例"
- 针对管理培训：强调"策略"、"思维"
- 针对通识培训：强调"趣味"、"易懂"

### 技巧 3: 模型选择
- **gemini-2.5-pro**: 最高质量，适合正式课程
- **gemini-2.0-flash-exp**: 最快速度，适合快速测试
- **gpt-4o**: OpenAI 用户的最佳选择

### 技巧 4: 配置管理
- 如果 API 出现问题，可以在侧边栏点击"修改配置"立即调整
- 如果想重新开始，可以点击"删除配置"清空所有设置

## ⚠️ 注意事项

### 配置文件位置
v2.0 的配置文件保存在：
- Windows: `C:\Users\你的用户名\.courseforge\courseforge_config.json`
- Mac: `/Users/你的用户名/.courseforge/courseforge_config.json`
- Linux: `/home/你的用户名/.courseforge/courseforge_config.json`

### API Key 安全
- API Key 存储在本地，不会上传
- 配置文件具有读写权限保护
- 建议定期更换 API Key

### 系统 Prompt 最佳实践
- 不要删除 `# Role`、`# Constraints`、`# Task Workflow` 标签
- 保持结构完整性
- 测试自定义 Prompt 后再正式使用

## 🐛 常见问题

### Q: 启动脚本卡住不动？
**A**: 可能是网络问题导致依赖下载慢。
- Windows: 编辑 `start.bat`，删除镜像源参数
- Mac/Linux: 编辑 `start.sh`，删除镜像源参数

### Q: API 测试失败？
**A**: 检查以下几点：
1. API Key 是否正确（应以 `sk-` 开头）
2. Base URL 是否以 `/v1` 结尾
3. 网络是否能访问 API 服务
4. 是否有剩余额度

### Q: 找不到配置文件？
**A**: 配置存储在用户主目录的隐藏文件夹中：
```bash
# Mac/Linux
ls -la ~/.courseforge

# Windows (PowerShell)
ls $HOME\.courseforge -Force
```

### Q: 想回到 v1.0 的配置方式？
**A**: 虽然不推荐，但你仍可以创建 `.streamlit/secrets.toml` 文件。
v2.0 会优先使用程序内配置，除非配置文件不存在。
